# Plan

1. Import the `Counter` class from the `collections` module.
2. Count the frequencies of all characters in the input string `s` using `Counter(s)`.
3. Iterate through each character `c` in the string `s`.
4. If the count of the character `c` in the frequency dictionary is `1`, return `c`.
5. If the loop completes without finding any non-repeating character, return `None`.
