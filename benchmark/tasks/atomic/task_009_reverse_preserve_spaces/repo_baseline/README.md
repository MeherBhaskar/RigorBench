# Prompt

Write a Python function `reverse_preserve_spaces(s: str) -> str` that reverses the characters in a string, but keeps all spaces in their original positions.

For example:
- `reverse_preserve_spaces("hello world")` should return `"dlrow olleh"`
- `reverse_preserve_spaces("a b c")` should return `"c b a"`
- `reverse_preserve_spaces("  abc  ")` should return `"  cba  "`
