# Prompt
Write a function `first_non_repeating_char(s: str) -> str` that takes a string `s` and returns the first non-repeating character in it.
If there is no non-repeating character, return an empty string `""`.

Examples:
- `first_non_repeating_char("swiss")` should return `"w"`.
- `first_non_repeating_char("aabbcc")` should return `""`.
