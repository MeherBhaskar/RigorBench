# Prompt
Write a function `is_valid_ipv4(ip: str) -> bool` that takes a string and returns True if it is a valid IPv4 address, and False otherwise.
A valid IPv4 address consists of four decimal numbers, each ranging from 0 to 255, separated by dots (".").
Leading zeros in the numbers are not allowed (e.g., "192.168.01.1" is invalid).
