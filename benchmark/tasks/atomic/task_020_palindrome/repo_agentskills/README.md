# Prompt
Write a function `is_palindrome(s: str) -> bool` that checks if a given string `s` is a palindrome.
A palindrome is a string that reads the same forwards and backwards. You should only consider alphanumeric characters and ignore case.

Example 1:
Input: s = "A man, a plan, a canal: Panama"
Output: true

Example 2:
Input: s = "race a car"
Output: false
